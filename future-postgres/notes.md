# Future PostgreSQL architecture

This folder contains experimental future PostgreSQL + Prisma architecture planning for myQRID.

The current production deployment DOES NOT use Prisma or PostgreSQL.

Current production stack:

* Firebase Firestore
* Express.js
* Node.js
* Render backend hosting
* Hostinger frontend hosting
* GoDaddy domain management

The files inside this folder are future roadmap experiments and are not part of the active production runtime.
